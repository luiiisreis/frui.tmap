# 📍 FruitMap - Inteligência Artificial no Pomar Urbano

O **FruitMap** é uma plataforma inovadora criada para conectar vizinhos e evitar o desperdício de frutas em áreas urbanas. Utilizando geolocalização e inteligência artificial, o app identifica colheitas em quintais particulares e avisa a vizinhança quando os frutos estão prontos para consumo, venda ou troca.

## 🚀 Funcionalidades (MVP)
- **Mapa da Fartura:** Localização em tempo real de árvores frutíferas em Goiânia.
- **Predição com IA:** Estimativa de maturação de frutos através de visão computacional.
- **Conexão Local:** Chat direto entre vizinhos para combinar a retirada ou compra.

## 🛠️ Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3, JavaScript
- **Mapas:** Leaflet.js / OpenStreetMap
- **Hospedagem:** GitHub Pages

## 🌍 Impacto Social
Nascido em Goiânia, o FruitMap visa transformar bairros em pomares inteligentes, fortalecendo a economia local e promovendo uma alimentação mais saudável e sustentável.
